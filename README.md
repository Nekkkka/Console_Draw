Build & Run
Requirements:

- Windows
- C++ compiler (e.g. MinGW, MSVC)
- C++17 or later

How to build:
- git clone https://github.com/username/Console_Draw.git
- cd ConsoleDraw
- g++ main.cpp -o Console_Draw
- ConsoleDraw.exe

(or open the project in Visual Studio / Code::Blocks)
